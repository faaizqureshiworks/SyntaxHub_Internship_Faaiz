# Syntecxhub Python Internship Portfolio 🚀

Welcome to my central repository for the **Syntecxhub Python Programming Internship**. This workspace contains all the weekly projects and assignments I completed during the program, showcasing my progression from basic scripting to building interactive Command Line Interface (CLI) applications and performing data analysis.

---

## 📂 Repository Structure

This repository is organized week-by-week, demonstrating a progressive enhancement of Python skills, including Object-Oriented Programming (OOP), file handling, data manipulation, and data visualization.

### [Week 1] - Python Fundamentals & Environment Setup
* **File:** `project_1.py`
* **Description:** Initial setup of the Python development environment, Git version control configuration, and foundational logic scripting.

### [Week 2] - Student Management System
* **File:** `student_management.py`
* **Description:** A complete CLI application to manage student records.
* **Key Skills:** * Object-Oriented Programming (Classes & Objects)
  * CRUD Operations (Create, Read, Update, Delete)
  * Data Persistence using `JSON` file handling.

### [Week 3] - CSV to Excel Data Converter
* **File:** `csv_to_excel.py`
* **Description:** An automated data-cleaning script that reads messy CSV files, sanitizes the text, handles missing data, and exports a professionally formatted `.xlsx` file.
* **Key Skills:** * Data Manipulation with `pandas`
  * Text normalization and duplicate removal
  * Excel integration using `openpyxl`.

### [Week 4] - Personal Expense Tracker CLI
* **File:** `expense_tracker.py`
* **Description:** A dynamic financial tracking tool that logs incomes and expenses to a CSV database, calculates net balances, and generates visual charts of spending habits.
* **Key Skills:** * CSV File I/O
  * Data Aggregation with `pandas`
  * Data Visualization using `matplotlib`.
